# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is enabled on this template. See [this documentation](https://react.dev/learn/react-compiler) for more information.

Note: This will impact Vite dev & build performances.

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.

# 🛒 Projeto: Loja de Informática

## 📌 Sobre o Projeto
Este projeto tem como objetivo desenvolver um site de uma **loja de informática**, aplicando os conceitos aprendidos nas últimas aulas.

A aplicação permitirá o cadastro e a exibição de produtos organizados por **seções** e **marcas**.

---

## 🎯 Objetivos
- Criar um formulário para cadastro de produtos  
- Organizar produtos por categorias  
- Exibir os produtos dinamicamente na página  

---

## 🧩 Funcionalidades

### 📂 Seções da Loja
Os produtos devem ser organizados nas seguintes categorias:

- 💻 Computadores  
- 🖱️ Acessórios  
- 🖨️ Impressoras  
- 🎮 Games  
- 📱 Gadgets  

---

### 🏷️ Marcas
Cada produto deve possuir uma marca:

- HP  
- Dell  
- Positivo  
- Asus  
- Xing Ling (genérico)  

---

## 📝 Formulário de Cadastro

O sistema deve conter um formulário com os seguintes campos:

| Campo        | Tipo            |
|-------------|-----------------|
| Seção       | Select (dropdown) |
| Marca       | Select (dropdown) |
| Nome        | Input texto      |
| Preço       | Input (texto ou número) |
| Estado      | Radio (Novo / Usado) |

### 🔘 Ação
- Botão: **Inserir Produto**

---

## 🖼️ Exibição dos Produtos

Os produtos cadastrados devem aparecer em formato de **card**, contendo:

- 🖼️ Imagem representando a marca  
- 📌 Nome do produto  
- 💰 Preço  
- 🔄 Estado (Novo ou Usado)  

---

## 💡 Regras do Sistema

- O usuário deve conseguir cadastrar produtos  
- Os produtos devem ser exibidos na tela após o cadastro  
- Os produtos devem ser organizados por seção  
- Cada produto deve mostrar corretamente suas informações   

---

## 🚀 Tecnologias Utilizadas

- HTML  
- CSS  
- JavaScript
- React

---

## ✅ Critérios de Avaliação

- ✔️ Funcionamento do formulário  
- ✔️ Exibição dos produtos  
- ✔️ Organização por seções  
- ✔️ Estrutura do código  
- ✔️ Interface visual  

---

## 📚 Aprendizados

Este projeto reforça conceitos de:

- Manipulação do DOM  
- Eventos em formulários  
- Organização de dados  
- Estruturação de páginas web  

---
